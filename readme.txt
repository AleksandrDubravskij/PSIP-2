BeautySalon - Система записи в салон красоты
Проект представляет собой веб-приложение для записи клиентов в салон красоты. Пользователи могут просматривать услуги, выбирать мастера и удобное время для записи. Авторизованные пользователи могут управлять своими записями через личный кабинет.

Основные функции
Просмотр услуг: Список всех доступных услуг с описанием и стоимостью.

Запись на услугу: Выбор мастера и времени для записи.

Личный кабинет: Просмотр и управление своими записями.

Авторизация: Регистрация и вход для клиентов.

Административная панель: Управление услугами, мастерами и записями (в разработке).

Установка и запуск
Требования
PHP 7.4 или выше

MySQL 5.7 или выше

Веб-сервер (Apache, Nginx)

Composer (для управления зависимостями)

Установка
Клонируйте репозиторий:

bash
Copy
git clone https://github.com/ваш-username/BeautySalon.git
cd BeautySalon
Установите зависимости:

bash
Copy
composer install
Настройте базу данных:

Создайте базу данных MySQL.

Импортируйте структуру из файла database.sql.

Настройте конфигурацию:

Создайте файл config.php на основе config.example.php.

Укажите параметры подключения к базе данных.

Запустите веб-сервер:

bash
Copy
php -S localhost:8000
Откройте проект в браузере:

Copy
http://localhost:8000
Структура проекта
assets/: CSS и JavaScript файлы.

config/: Конфигурационные файлы.

includes/: Подключаемые файлы (шапка, подвал, настройки БД).

pages/: Основные страницы приложения.

database.sql: SQL-файл для создания структуры базы данных.

Используемые технологии
PHP: Основной язык серверной части.

MySQL: База данных для хранения информации.

HTML/CSS: Верстка и стилизация.

JavaScript: Интерактивные элементы (календарь, динамическое обновление).
